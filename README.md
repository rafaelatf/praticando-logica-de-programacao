## 🎯 Objetivo

Fortalecer o raciocínio lógico e colocar em prática os fundamentos da programação com **HTML**, **CSS** e **JavaScript**, por meio de desafios interativos e dinâmicos.

---

## 💻 Projetos Desenvolvidos

### 🎲 **Sorteador de Números**
Ferramenta interativa que sorteia números aleatórios dentro de um intervalo definido pelo usuário.  
Conceitos aplicados: `for`, `parseInt`, funções, manipulação de DOM e template strings.

## 🧠 Conceitos praticados

- Variáveis e tipos de dados  
- Estruturas condicionais (`if/else`, `switch`)  
- Estruturas de repetição (`for`, `while`)  
- Manipulação do DOM (`getElementById`, `querySelector`)  
- Arrays (`push`, `includes`, `length`)  
- Funções e modularização  
- Template Strings  
- Manipulação de classes CSS (`classList.add`, `remove`, `contains`)  
- Eventos e interatividade com o usuário
